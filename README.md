# Fermata Coming Soon Site

Static temporary site for Fermata, a music intelligence lab.

## Files

- `index.html` - the page to deploy
- `styles.css` - all styling
- `assets/fermata-blueprint.png` - custom visual asset
- `assets/favicon.png` - browser icon

## Deploy

Upload the full contents of this folder to the domain root. No build step or server runtime is required.
